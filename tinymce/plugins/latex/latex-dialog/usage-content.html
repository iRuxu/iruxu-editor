LaTeX 公式编辑器使用指南

1. 行内公式 （文字中的小公式）

在文字中插入小型公式，有两种写法：

方法一：使用 $ 单个美元符号
  • ⌨️ 示例：勾股定理 $a^2 + b^2 = c^2$
  • 🎉 效果：$a^2 + b^2 = c^2$
方法二：使用 \(\) 括号
  • ⌨️ 示例：爱因斯坦质能方程 \(E = mc^2\)
  • 🎉 效果：\(E = mc^2\)

2. 独立公式 （大型公式块）

对于需要突出显示的重要公式，使用块级显示：

方法一：使用 $$ 双美元符号
  • ⌨️ 示例：
$$
y = \sqrt{x_1^2 + x_2^2 + x_3^2 + x_4^2}
$$
  • 🎉 效果：
$$y = \sqrt{x_1^2 + x_2^2 + x_3^2 + x_4^2}$$
方法二：使用 \[\] 括号
  • ⌨️ 示例：
\[
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
\]
  • 🎉 效果：
\[\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}\]
💡 提示：块级公式会自动居中显示，适合长公式或多行公式。

3. 源码指定（高级）

如果您精通 HTML，可以直接通过编辑源码，对包裹区使用特定的类 .w-latex，系统会自动处理换行和格式。

  • ⌨️ 示例：
<div class="w-latex">
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
</div>
  • 🎉 效果：
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
